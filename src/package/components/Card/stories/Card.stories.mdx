# Card

This is a simple container with a shadow that can be used to
create card-like elements. It includes the Flexbox, Layout, Space, and
Typography utilities from [styled-system][1].

![]()

## Usage

```tsx
import { Card } from "hazel-ui";

function Component() {
  return (
    <Card display="flex" justifyContent="space-between">
      <div>1</div>
      <div>2</div>
    </Card>
  );
}
```
